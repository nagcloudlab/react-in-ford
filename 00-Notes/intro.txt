
UI = data + style + interaction
Web UI = HTML + CSS + JS


why we need javascript?

1. to build dynamic web-pages
2. to build sever-side applications
3. to build mobile applications
4. to build desktop applications
5. to build command-line applications

javascript features:

1. dynamic typing
2. functions as first-class citizens
3. single-threaded 
4. all IO operations are async ( non-blocking )


javascript is based on ECMAScript standard
latest : ES6 ( ECMAScript 2015 )


how to run javascript code?

=> javascript engine / interpreter

1. browser envi

    1. chrome - v8 ( c++ )
    2. firefox - spidermonkey ( c++ )
    3. safari - nitro ( c++ )
    4. edge - chakra ( c++ )

2. other envi

    1. node.js - v8 ( c++ )
    2. JVM - nashorn ( java )
